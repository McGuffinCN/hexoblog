支持自定义背景的百度/谷歌CSS美化样式

发表于 2019-05-13 分类于 折腾 阅读: 评论:

结合了AC-baidu和一些自己写的CSS，外加一些去广告功能，让你的搜索引擎与众不同。

百度效果图

谷歌效果图

安装地址

百度结果页美化 百度首页美化 谷歌全局美化

使用方法

  • 安装为CSS样式(推荐)
  • 安装为油猴脚本

1，安装一个CSS美化扩展，由于Stylish曾爆出窃取用户隐私，故推荐Stylus

浏览器 可选的扩展
Chrome Stylus 或 Stylish
Firefox Stylus 或 Stylish
其他浏览器 请在各自的扩展中心内搜索

2，(可选)在样式页面选择Customize Settings自定义背景图或填充颜色

3，Install Style 安装脚本，完成

1，安装一个用户脚本管理器

浏览器 可选的扩展
Chrome Tampermonkey Violentmonkey
Firefox Greasemonkey Tampermonkey Violentmonkey
Safari Tampermonkey
Microsoft Edge Tampermonkey
Opera Tampermonkey Violentmonkey
其他浏览器 请在各自的扩展中心内搜索

2，样式页面选择Install style as userscript

3，（可选）可在background属性下自定义背景图片
