PS4平台自动化工具PS4 Marco

发表于 2020-04-04 分类于 游戏 阅读: 评论:

最近在玩《Granblue Fantasy: Versus》，因为要获得RPG模式Hard难度通关赠送的特典码，所以被迫当一回骑空士刷图农武器。所幸RPG模式有自动战斗省去不少麻烦，不过每次结算后还是需要手动续战，所以找了个可以在PS4使用宏的工具PS4 Macro来帮我自动输入续战指令，这样就可以挂着全自动种地睡觉去了。

GitHub项目地址 下载地址

以下使用指南大部分直接翻译自GitHub页面ReadMe。

使用

要使用PS4 Macro，首先要有一台可以运行PS4 Remote Play的电脑。连接成功后运行PS4 Macro。

电脑插上DS4手柄后，按下触摸板可以开始录制指令，再次按下将结束录制并自动循环播放。
如果你需要录制包括触摸板的指令，请将 Playback->Record On Touch 选项取消勾选，点击RECORD（热键Ctrl+R）然后点击PLAY来开始录制，再次点击RECORD（Ctrl+R）以结束录制。

NOTE: 如果要在没有连接手柄的情况下工作，请见下面设置中的EmulateController选项。

设置

你需要在根目录创建一个settings.xml文件来写入设置项。

设置项 描述 默认值
AutoInject 自动轮询PS4 Remote Play并在可能的情况下进行注入 false
BypassInjection 为了调试而绕过注入 false
EmulateController 使用模拟手柄运行（不使用手柄） false
ShowConsole 启动时打开调试控制台 false
StartupFile 启动时加载的文件的绝对或相对路径（可以是xml或dll） null

重映射

Remapper可以将手柄按键映射到键盘，使用键盘来操作PS4游戏。

要添加录制好的宏，请单击…浏览并选择一个xml宏文件。

其它高级操作就不翻译了，想深入研究的去GitHub吧。

替代方案

如果你由于种种原因无法使用PS4 Macro，这里有个替代方案。那就是使用夜神模拟器运行安卓版PS4 Remote Play，然后使用夜神模拟器的操作录制功能完成自动化。
