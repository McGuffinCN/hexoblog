我的Windows软件清单

发表于 2020-03-10 分类于 软件 阅读: 评论:

给目前已安装的软件列个清单，以便将来重装系统或换电脑时参考。

工具

软件名 描述
ADWCleaner 广告清理工具，强力到可能导致QQ崩溃
CrystalDiskInfo 查看硬盘健康信息
Everything 最好最快的Windows文件搜索工具
FaceRig 迫真虚拟主播
FileOptimizer 文件瘦身，支持多种格式
oCam 录屏软件，可以直接录制为Gif，缺点是每次关闭会弹出网页
Poedit 用于WordPress主题或插件的翻译
PotPlayer 最强播放器
Process Viewer 定位屏幕上窗口的进程，比如弹窗广告
Sandboxie 沙盒，让软件在隔离环境运行
Scrcpy 简易的手机投影到电脑软件，无需Root
SmoothVideo Project 视频自动补帧工具，经常出现音画不同步
Visual Studio Code 一款文本编辑器，强大美观
Sublime Text 又一款文本编辑器，简洁美观
Notepad ++ 双一款文本编辑器，强大粗糙
UltraEdit 叒一款文本编辑器，我怎么装这么多
EmEditor 叕一款文本编辑器，忘掉他吧
UltraISO 虚拟光驱
Winamp 鬼知道我为什么还在用这个老古董
FormatFactory 大名鼎鼎的格式工厂

网络

软件名 描述
Aria2 轻量级的多协议和多源命令行下载程序，Web端控制台
BrookTools 伟大的doubi站长写的Brook客户端，让我们缅怀他
eMule 电驴
Fiddler HTTP调试抓包工具，记录并检查所有电脑和互联网之间的HTTP通讯
FlashFXP FTP工具，用于与服务器交换文件
JiJiDown 下载批哩批哩的视频和音频
PanDownload 百度网盘下载工具，不好使了
PixivBiu 通过爬虫自动按收藏数排名Pixiv搜索结果
PuTTY Telnet、SSH、rlogin、纯TCP以及串行接口连接软件
SpeedPan 速盘，Aria2封装，用于下载百度网盘文件
SSTap 利用虚拟网卡实现的网络层代理，用来加速游戏
I2P Invisible Internet Project,俗称大蒜路由，改良自Tor
μTorrent Torrent种子下载工具，可以在线播放，Web端控制台
XAMPP 建站集成软件包，用于本地建站并测试
XX-Net 和GoAgent一样，时代的眼泪
YunDownload 又一个勇于与百度网盘搏斗的勇士，可惜凉了

创作

软件名 描述
3ML Editor MIDI播放与制作
Aerialod 简易地形生成工具，可以拖入图片自动生成地形
FL Studio 以乐段为基础的编曲软件，俗称水果（我怎么觉得是萝卜）
MarkdownPad Markdown编辑器，这篇文章就是用这个写的
MixMeister BPM Analyzer 自动分析音乐的BPM（每分钟节拍数）
Mocha Pro 摄像机反求平面跟踪软件，记住没激活前不要使用
PoseStudio 画画时用来参考人体结构
PaintTool SAI 画画用的简易版PS
Ulead GIF Animator Gif动图编辑工具
Waves 行业标杆混音效果器，虽然我只拿来做鬼畜

学习

软件名 描述
AutoCAD 傻逼软件
CAJViewer 知网指定CAJ阅读器
Lumion 3D渲染工具，可以与SketchUp同步显示
OneNote 微软的笔记工具
Revit BIM工具之一
SketchUp 艹图大湿
天正建筑 AutoCAD的本地化插件
Twinmotion 虚幻引擎3D渲染工具，简单但效果有限
XMind ZEN 思维导图生成
爱福窝云设计 室内设计并生成效果图
酷家乐 同上
建标库 建筑相关规范查询

游戏

软件名 描述
DeSmuME 最好的NDS模拟器
FCEUX 专业的NES/FC模拟器，可以编辑或播放TAS

MAME 街机模拟器，集成全套BIOS
No$GBA NDS/GBA模拟器
PPSSPP PSP模拟器
UsbEAm Hosts Editor 通过修改Hosts加速游戏平台
VirtuaNES 老派NES/FC模拟器
VisualBoyAdvance-M GBA模拟器
WeMod 游戏作弊器集成（不要打我，这是本地游戏的）
WinKawaks 街机模拟器，支持CPS1、CPS2和NEOGEO
您的支持是我创作的动力！
